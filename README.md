# Advanced Hand Tracking AR – README

## Project Overview
Advanced Hand Tracking AR (Neon Aura AR) is a browser-based Augmented Reality web application that uses MediaPipe Hands and the Web Audio API to create an interactive hand-tracking experience. Users can control visual effects, particle systems, shockwaves, neon animations, and sound effects using real-time hand gestures captured through their webcam.

## Features

### Hand Tracking
- Real-time hand detection using MediaPipe Hands
- Supports tracking up to 2 hands simultaneously
- Detects 21 landmarks per hand

### Gesture Recognition
- Pinch gesture detection (thumb + index finger)
- Open hand and fist recognition
- Hand spread percentage calculation

### Visual Effects
- Neon glowing hand skeletons
- Dynamic particle effects on fingertips
- Shockwave effects on pinch gestures
- Lightning effects between hands
- Rotating mandala visualization
- Matrix rain animated background
- Motion blur and trail effects

### Audio Effects
- Dynamic ambient hum sound
- Interactive sound modulation based on hand distance
- Electric zap sound when pinch gestures occur

### Themes
- Rainbow
- Cyberpunk
- Lava
- Ocean
- Galaxy

### HUD Display
- Hands Detected
- FPS Counter
- Gesture Status
- Hand Spread Percentage

## Technologies Used
- HTML5

## Installation
1. Download the project file.
2. Save it as `hand.html`.
3. Open it in a modern browser.
4. Allow camera permissions.
5. Click **Enter Experience**.
